# API 文檔

### Base URL

```
https://exptech.com.tw/api
```

## API 版本

`https://exptech.com.tw/api/v{版本號}`

### 速度限制
- **每秒** 最高 `50 RPS`
- **10分鐘** 最多 `10000 RPS`

### 所有 API 版本

| 版本 | 狀態   |
| ---- | ------ |
| 0    | 可用   |
| 1    | 製作中 |
